Ladies First

Website built for Ladies First a cappella.

This was a straightforward site built for the a cappella group Ladies First. The site pulls information from a database using PHP and MySQL. I also built a content management system for the future PR Manager of the group so that no coding skills were required to update the information.

The site is not yet responsive or adaptive, but that's a project for another day.
